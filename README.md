# AI Study Assistant

An AI-powered study assistant that allows users to upload study materials and interact with them through a document-based chatbot.

## Features

- Upload PDF documents
- Generate document summaries
- Ask questions based on uploaded content
- Get simplified explanations of topics
- Context-aware responses using AI

## Problem Statement

Students often spend significant time reading lengthy notes and study materials. This project helps students quickly understand content by providing summaries, explanations, and question-answering capabilities through an AI-powered interface.

## Project Goal

To transform static study materials into an interactive learning experience using artificial intelligence.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### AI Integration
- OpenAI API (or equivalent LLM)
